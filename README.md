## Sobre o Projeto

Este projeto reúne uma série de exercícios introdutórios de lógica de programação utilizando a linguagem Python. 

## Exercícios incluídos

1. **Número positivo ou negativo**: Solicita um número inteiro e informa se ele é positivo, negativo ou zero.  
2. **Par ou ímpar**: Verifica se o número informado é par ou ímpar.  
3. **Maioridade**: Recebe a idade de uma pessoa e informa se ela é maior ou menor de idade.  
4. **Nota de aprovação**: Avalia a nota de um aluno (de 0 a 10) e indica se foi aprovado ou reprovado.  
5. **Comparação de dois números**: Compara dois números inteiros e informa qual é maior ou se são iguais.  
6. **Desconto em produto**: Aplica 10% de desconto em produtos com valor acima de R$100; caso contrário, mantém o preço original.  
7. **Login simples**: Simula um sistema de login que permite acesso apenas se o usuário for "admin" e a senha "1234".  
8. **Par ou múltiplo de 5**: Verifica se um número é par, múltiplo de 5 ou nenhum dos dois.  
9. **Conversão de temperatura**: Classifica uma temperatura em Celsius como “Congelante”, “Agradável” ou “Quente”.  
10. **Calculadora simples**: Realiza operações básicas (+, –, *, /) entre dois números inteiros com base na escolha do usuário.
